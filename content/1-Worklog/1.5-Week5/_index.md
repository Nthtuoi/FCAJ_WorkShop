### Week 5 Achievements:

* Understanding the nature of Amazon S3 object storage services:

  * Data storage: Mastering the principles of unstructured data storage through the concepts of Buckets, Objects, and cost optimization methods based on the data lifecycle (Lifecycle Policy/Storage Classes).

  * Static Website: Understanding the mechanism of Static Website Hosting to transform an S3 Bucket into storage space and run a static website directly on the Internet.

  * ...

* Practicing S3 configuration and handling permission errors (Hands-on S3 Labs):

  * Static Website Deployment: Successfully uploading source code (.html, .css, .js) and enabling Endpoint URLs for static websites. Identifying and isolating 403 Forbidden errors when the system has not been granted permissions.

  * Security Configuration: Master access management skills by configuring S3 Block Public Access and writing JSON code for the S3 Bucket Policy (s3:GetObject), effectively handling permission errors and ensuring smooth public website operation.

  * ...

* Accessing Amazon Lightsail Rapid Deployment Solutions:

  * Solution Optimization: Clearly differentiate the scope of use between Lightsail (fixed cost, all-inclusive) and EC2 (flexible, highly customizable); understand the optimal cost-effectiveness and time-to-market approach for small projects.

  * WordPress Practice: Successfully create a WordPress instance with a built-in cloud database; proficiently set up Static IP and configure a firewall directly on the simplified Lightsail interface.
